# 🟣 Midstacksec — Portfolio

Portfólio pessoal desenvolvido com Next.js 14, TypeScript e CSS Modules.

A ideia aqui não foi só montar um site bonito, mas criar algo com identidade visual própria, e explorar efeitos mais interativos sem depender de bibliotecas prontas.

---

O que tem aqui?

* Usei Next.js 14 com App Router
* TypeScript pra organização e tipagem
* CSS Modules (sem UI lib)
* Cursor customizado com glow
* Partículas no background usando canvas
* Efeito glitch no nome
* Terminal com digitação simulada
* Animações de scroll
* Contadores animados
* Layout responsivo

---

Evitei usar bibliotecas prontas de propósito. Quis ter mais controle sobre os efeitos e entender melhor como cada parte funciona.

Algumas coisas (como o terminal e o glitch) foram feitas manualmente pra treinar lógica, timing e manipulação de estado.

---

Stack

* Next.js

* CSS Modules
